## 📥 How to Clone and Run this Project

If you would like to run this 3D interactive portfolio locally on your machine, simply follow these steps:

### 1. Clone the Repository
Open your terminal and run the following command to clone the project:
```bash
git clone https://github.com/sankar069/my-portfolio.git
```

### 2. Navigate to the Project Directory
```bash
cd my-portfolio
```

### 3. Install Dependencies
You will need Node.js installed on your machine. Run the Node Package Manager (npm) to install all the required packages:
```bash
npm install
```

### 4. Start the Development Server
Run the vite development server:
```bash
npm run dev
```

### 5. View in Browser
Open your browser and navigate to the local host link provided in your terminal (usually `http://localhost:5173/`).
